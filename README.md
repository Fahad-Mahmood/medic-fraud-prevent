## Setup

Install dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

## Run Development Server

Start the development server on `http://localhost:8080`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```
